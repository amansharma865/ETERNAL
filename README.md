# Eternal Watches

**Eternal Watches** is a premium motion-driven showcase website built to highlight luxury timepieces with elegance and interactivity. Crafted using **HTML**, **CSS**, **JavaScript**, and **GSAP**, it focuses on smooth transitions, parallax animations, and immersive storytelling to deliver an engaging user experience.

## Features
- **Motion-Driven UI:** Dynamic scroll and hover animations powered by GSAP.  
- **Smooth Transitions:** Seamless page and element animations for a premium look.  
- **Luxury Product Showcase:** Displays high-end watches in an elegant layout.  
- **Interactive Experience:** Subtle animations that respond to user interactions.  
- **Responsive Design:** Fully optimized for desktop, tablet, and mobile.  
- **Brand Story Section:** Tells the story behind the brand with a cinematic flow.  
- **Buy Now Page:** Basic e-commerce-style product display for demonstration.

---

## How It Works

1. **Landing Page:**  
   - Introduces the Eternal Watches brand with smooth hero animations.  
2. **Product Showcase:**  
   - Highlights featured watches with motion-based transitions.  
3. **About Section:**  
   - Presents the brand story and philosophy through scrolling effects.  
4. **Product Page:**  
   - Displays detailed information and “Buy Now” options for selected models.  
5. **Responsive Layout:**  
   - Automatically adjusts content for any screen size or orientation.

---

## Tech Stack
- **Frontend:** HTML5, CSS3, JavaScript (ES6)  
- **Animations:** GSAP (GreenSock Animation Platform)  
- **Design:** Flexbox, CSS Grid, Parallax effects  
- **Deployment:** Netlify  

---

## Usage Guide

Scroll through the site to experience motion-driven animations.

Hover over watches to see subtle interactive effects.

Explore brand sections for visual storytelling.

Click “Buy Now” buttons to experience the mock e-commerce flow.
